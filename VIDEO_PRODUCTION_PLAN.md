## 3. 详细分镜脚本 (Script)

### Part 1: Intro
> (画外音): "你是否也曾面对过这样杂乱无章的下载文件夹？文档、图片、安装包混在一起，找文件如同大海捞针。今天，我将向大家介绍我开发的 Python 文件管理工具集——一个包含智能整理和极速搜索功能的桌面应用解决方案。"

### Part 2: File Organizer Demo
> (画外音): "首先是核心组件——智能文件整理器。我们直接运行 `main.py` 启动现代化 GUI。"
> (操作): 选择测试文件夹。
> (画外音): "大家请看，这个文件夹里混合了各种文件和子文件夹。点击 'Start Organize'。"
> (操作): 点击按钮，弹出成功提示。
> (画外音): "瞬间完成。注意看，程序自动创建了 Documents, Images 等分类。**更重要的是**，根据最新的功能更新，所有的快捷方式 (.lnk) 和应用程序 (.exe) 被优先识别并隔离到了 Shortcuts 和 Applications 目录中，互不干扰。"
> (画外音): "同时，程序采用了**非递归扫描**模式，你可以看到这个 'SubDir' 子文件夹完好无损，内部结构未被破坏。这对于维护项目目录结构至关重要。"

### Part 3: Search Demo
> (画外音): "接下来是文件搜索专家。不同于系统搜索的缓慢，它支持实时预览。"
> (操作): 搜索关键字，点击结果。
> (画外音): "选中文件，右侧直接预览内容，无需打开外部程序。"

### Part 4: Technical Deep Dive
> (画外音): "现在让我们深入代码层面。本项目基于 Python 标准库开发，无需繁重的第三方依赖。"
> (切换到 VS Code): 打开 `src/classifier.py`。
> (画外音): "在 `classifier.py` 中，我实现了一套**三级优先级分类算法**。代码优先检查 `special_types`（如快捷方式），其次是用户定义的关键字，最后才是文件扩展名。这确保了特殊文件不会被错误的归类。"
> (切换到 `src/organizer.py`)。
> (画外音): "在文件扫描环节，我使用了高效的 `os.scandir` 迭代器，并显式过滤了目录条目，实现了非递归处理。同时，配合 `os.path.exists` 检查实现了自动重命名机制，杜绝了文件覆盖的风险。"

### Part 5: Outro
> (画外音): "这个项目不仅解决了实际问题，也展示了 Python 在 GUI 开发、文件系统操作和算法设计上的最佳实践。欢迎访问 GitHub 查看源码。谢谢观看！"

## 4. 准备工作清单 (Checklist)

### 环境准备
- [ ] **测试数据**: 准备一个包含 `.lnk`, `.exe`, `.jpg`, `.docx` 以及子文件夹的 "MessyFolder" 用于演示。
- [ ] **IDE 设置**: VS Code 设置为高对比度主题，放大字体 (Zoom Level 1.2+)，隐藏侧边栏以减少干扰。
- [ ] **重置环境**: 确保 `config.json` 为默认状态，删除之前的日志文件。

### 录制工具
- [ ] **OBS Studio**: 设置分辨率为 1920x1080 (1080p)，帧率 30fps 或 60fps。
- [ ] **麦克风**: 测试收音清晰，无明显底噪。

- [ ] 确保 `TECHNICAL_DETAILS.md` 已完成，可作为视频中展示文档的一部分。
